# Autonomyx — Primitives (captured)

**DRAFT · unsigned.** Provenance: the founder (@fractionalpm). Honesty note: Cortex, Temporal,
and Heart were first described **metaphorically** in-session ("Watcher / temporal lobe /
heart"), which the founder later flagged as the *"consider X as Y"* (imagination) problem.
They are captured here **for the record** and **must be redefined with real referents** before
they enter the enforceable spec. **Heartbeat** is now a **real component** (Apache HertzBeat).

## Heartbeat — Apache HertzBeat (REAL)
- **Source:** `https://hertzbeat.apache.org/` (egress-blocked — capabilities to be pinned from
  HertzBeat's own docs, not agent memory).
- **Role:** the heartbeat / monitoring / observability layer — produces the liveness and
  health evidence the system runs on. This makes "the heartbeat is the evidence" a real,
  buildable component (an existing Apache open-source tool) rather than a metaphor.
- **Pin note:** record a specific HertzBeat version + dated docs snapshot before use.

## Cortex (metaphor — needs real redefinition)
- As described in-session: the "Watcher" — conscious oversight/deliberation of a turn; judges
  only what it has watched (guard-before-act, screen-on-output, exit-shield).
- **To redefine for real:** likely the agent's per-turn control/deliberation component (the
  loop's reason→guard→act→screen step), stated without the mind metaphor.

## Temporal (metaphor — needs real redefinition; note name clash)
- As described in-session: the "temporal lobe" — append-only, time-ordered evidence / episodic
  memory; powers replay and recovery.
- **Name clash to resolve:** "Temporal" is also a real durable-execution system (temporal.io)
  and the gold standard for deterministic replay. Decide whether Autonomyx's time/evidence
  layer *is* that product, or a distinct component, before pinning the name.

## Heart (metaphor — not a spec term)
- As described in-session: intent / validity — the human who means and signs. This is **vision
  language**, not a contract term. In the enforceable spec it is simply: **validity is a human
  signature; the Operator is the accountable Actor.** "Heart" stays in the vision/brand, not
  the contract.
