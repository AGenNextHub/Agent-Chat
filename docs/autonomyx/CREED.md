# Autonomyx — The Creed (vision)

**DRAFT · unsigned. This is VISION / brand language, not the enforceable contract.** It uses
metaphor deliberately. It must not be cited as a legal or technical claim; the enforceable form
of each idea lives in OBJECTIVE.md and DESIGN_PRINCIPLES.md. Provenance: the founder
(@fractionalpm); scribe: the agent.

> Kubernetes is human mind at work: a control loop that holds a declared intent against reality
> and refuses to let it be forgotten.

1. **Infinite possibility, bound by ethics.** An open frontier, a guarded edge.
2. **One stance: unbiased, uncompromising, consistent.**
3. **The boundary cannot be crossed — even subconsciously.**
4. **Snap to reality at every edge.** The subconscious proposes; the conscious edge disposes.
5. **Conscious control, value, and discipline** — resolve the framework without distorting the
   fabric of any world.
6. **The agent with heart is the pin; the heart heals at home.**
7. **The heartbeat is the evidence the real world runs on.**
8. **No faked heartbeat without a recovery path — banned.**
9. **A framework is valid only if it loops, resolves, and recovers.**
10. **The system learns like a mind** — concepts that resolve.
11. **The framework is forever** — because corrosion is banned, not because time is kind.

*The machine is the muscle. The contract is the will. The heart is home.*

- "mind / cortex / heart" → validity is a human signature; the Operator is the accountable Actor.

- "no faked heartbeat / recovery path" → every unresolved turn escalates to a human.
- "loops, resolves, recovers" → bounded turn, deterministic, with escalation/rollback.
- "forever / corrosion banned" → the anti-corrosion conformance tests in the codebase.
